How To Run PHP Programs
-----------------------
- To validate that the PHP interpreter is accessible via VS Code terminal
    - Open terminal (CTRL + `)

    - Type the command

        php -v

    - PHP interpreter returns the following:

        PHP 8.2.12 (cli) (built: Nov 25 2023 08:09:53) (NTS)
        Copyright (c) The PHP Group
        Zend Engine v4.2.12, Copyright (c) Zend Technologies


- To execute a PHP program on the terminal
    - Use the following convention

        php <program_name>.php

    - Press ENTER to execute the program
